# Harsha's Python Notes — Webpage

A small, panda-themed landing page for tracking beginner-friendly Python notes, examples, and practice programs.

## Preview

![Python Notes webpage screenshot](images/webpage/screenshot.png)

## About

This page is a lightweight index for the [Python notes repository](https://github.com/harshada137/Python). It highlights the topics covered so far and links out to the full repo for details.

**Topics currently covered:**
- Modules
- Import statements
- Recursion
- Variable scope

## Tech

- Plain HTML/CSS, no build step or dependencies
- Fonts: [Baloo 2](https://fonts.google.com/specimen/Baloo+2), [Inter](https://fonts.google.com/specimen/Inter), [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) (via Google Fonts)
- Icons: [Font Awesome](https://fontawesome.com/)

## Usage

Open `python-notes.html` directly in a browser — no setup required.
