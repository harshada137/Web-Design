# 🌸 Cute Landing Page

A simple and aesthetic landing page built using **HTML**, **CSS**, and a little **JavaScript**. It features a soft pastel theme, floating emojis, glassmorphism effects, and a responsive design.

## ✨ Features

- 🌈 Pastel gradient background
- 💎 Glassmorphism card
- 💖 Floating emoji animations
- 🎀 Interactive button
- 📱 Responsive layout
- ⚡ Single-file project (HTML, CSS, and JavaScript combined)
